一些想法

今天广东的天气不太好。
我坐在往常的书桌前，掂量着时间，等到网课开始。
这篇文章，就是在这个短小的空闲时间中写出来的
这篇文章的产生，不算容易，中间稿件不知为什么，损坏了一次，也算是比较曲折吧。

Hello, Github!

这句话，是我刚刚进入Github的第一天，写下的第一行Markdown。
我从入 Github 到现在，从一个编程小白，成长为了一个略懂皮毛的 半码农。
我发现 Github 确实是一个宝库，上面的开源项目，能学到很多某某课上没讲到的知识和经验 (这话我没有贬低的意思，请不要误会)。
于是，我慢慢的开始尝试着为开源社区做点贡献。
而我现在也在 Working On Progress ，开始和其他人合作，写一些项目。
而随着项目的开发,或许我会发现，将来的某一天，我会将开源真正的当做自己的信仰吧。
